printf() is an output function in c language. which is used to print the output to the console

it is available in stdio.h library.

Syntax:
    printf("format Specifier",variable_name);

Ex:
    if you are taking an integer input 
            int num=100;
            printf("%d",num);

while printing the output.

    int num;
